# Pinstack
Chrome同期でピン留めタブのグループを保存・復元する拡張機能です。

## 特徴
- ピン留めタブを名前付きのグループとして保存
- デフォルトグループを新規ウィンドウで自動復元
- 端末間でグループを同期
- ポップアップから作成・削除・既定設定が可能
- 別端末から同期された更新を通知

## 使い方
1. 現在のウィンドウでピン留めタブを用意します。
2. 拡張アイコンのポップアップでグループ名を入力し保存します。
3. 一覧からデフォルトにしたいグループを選択します。
4. 新しいウィンドウを開くと自動的に復元されます。
